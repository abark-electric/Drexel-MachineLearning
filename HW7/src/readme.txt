To run the SVM script, please open Matlab, switch to the folder contain both SVM.m file and spambase.data file.

To run the Muti-class SVM script, please open Matlab, switch to the folder contain both MSVM.m file and CTG.csv file.
This script requires MATLAB® R2016b or later for in script functions.
Notice that I use “mode” function to select class beat other most, which will select the smallest one if an tie occur.
